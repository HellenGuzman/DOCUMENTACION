DICCIONARIO SOBRE EL DOCUMENTO WEB

En este apartado estan las palabras tecnicas que encontremos en el documento.

B:

.Barra de direcciones

Espacio del navegador donde se escribe la URL de un sitio para acceder a una página web.

Boton

Elemento interactivo en la interfaz que permite ejecutar una acción al presionarlo,
como enviar información o iniciar una herramienta.

C

4.Check

Marca o indicador visual que confirma que una acción fue completada o
que una opción fue seleccionada correctamente.

5.Click o Cliquear

Acción de presionar un botón del mouse o tocar la pantalla para interactuar con elementos de la web.

Conservacion de conversacion con la IA

Función que guarda el historial del chat para poder revisarlo o continuarlo más tarde.

Copia y pega

Acción de duplicar un texto o contenido y colocarlo en otro lugar sin necesidad de escribirlo nuevamente.

G:

Grabacion de audio

Herramienta que permite registrar la voz del usuario para enviarla a la IA y que esta la procese o responda.

H:

Herramientas

Conjunto de funciones disponibles dentro de la aplicación que facilitan el uso de la inteligencia artificial.

I:

Inteligencia artificial

Tecnología que permite a un sistema imitar capacidades humanas como comprender, aprender y generar respuestas.

N:

Navegador web

Programa o aplicación que permite acceder a internet y visualizar páginas web (por ejemplo, Chrome o Firefox).

O:

Opcion premium

Plan de pago que ofrece funciones avanzadas o beneficios adicionales respecto a la versión gratuita.

P:

Plataforma

Entorno digital donde se ofrece un servicio, como una web o aplicación para interactuar con la IA.

Q:

Quiz

Cuestionario o conjunto de preguntas que evalúan el conocimiento del usuario dentro de la plataforma.
